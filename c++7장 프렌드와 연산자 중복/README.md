3. 연산자함수란무엇인가?
: 연산자 함수는 특정 연산자를 재정의하기 위해 사용하는 함수입니다. 이 함수는 특정 타입의 객체 간에 연산을 수행하도록 정의되며, C++에서는 일반적으로 operator 키워드를 사용하여 정의합니다. 연산자 함수는 클래스의 멤버 함수 또는 전역 함수로 구현할 수 있습니다.
